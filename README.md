# 1. Project Title
Customer Shopping Behavior Analytics Dashboard 

## 2. Short Description
Comprehensive analytics platform revealing deep customer insights for e-commerce optimization—tracking subscription patterns (27% active), revenue drivers ($59.76 avg), demographic behaviors (Female 73%), category affinities, age-based trends, and satisfaction metrics (avg rating 3.8) to fuel personalized marketing and retention strategies.

## 3. Tech Stack
ETL & Processing: Python (Pandas), Jupyter Notebook, Power Query for advanced transformations  
Database & Querying: PostgreSQL (pgAdmin4), SQL for complex joins/aggregations  
Visualization & Modeling: Power BI (full-suite: DAX measures, data modeling, star schemas, relationships, slicers, drill-throughs)  
Version Control: Git/GitHub  

## 4. Data Source
Rich e-commerce customer dataset from PostgreSQL database (demographics, purchases, subscriptions, reviews), extracted via pgAdmin4/SQL, transformed/cleaned in Python Pandas/Jupyter (outlier removal, feature engineering), then semantically modeled in Power BI for interactive exploration.

## 5. Features and Highlights
Core KPIs: Subscription status (27% Yes vs 73% No), avg revenue per user ($559.76), customer count (3.9K), review ratings (3.8 avg)—all with custom DAX calculations  
Visual Analytics: Pie charts for gender (Female 73%, Male 27%), bar graphs for category sales (Apparel/Footwear leading), age-group revenue/sales heatmaps, donut charts for subscription splits  
Advanced Insights: Cross-filters linking demographics to behaviors, age-segmented performance (revealing high-value groups), trend analysis across categories, what-if scenarios for retention impact  
Scalability: Enterprise-ready with row-level security, mobile-optimized layout, real-time refresh capabilities, handling 10K+ rows efficiently
